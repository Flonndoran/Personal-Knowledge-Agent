# Personal Knowledge Agent 🧠

Персональный менеджер знаний с использованием MCP.

## 🚀 Быстрый старт

1. Активируйте виртуальное окружение:
   \\\powershell
   venv\Scripts\Activate.ps1
   \\\

2. Запустите CLI:
   \\\powershell
   python -m src.cli.main --help
   \\\

## 📁 Структура

- \src/\ - исходный код
- \config/\ - конфигурация
- \data/\ - данные
- \	ests/\ - тесты

## 🛠 Технологии

- Python 3.9+
- FastAPI (MCP серверы)
- Typer (CLI)
- Qdrant (векторная БД)
